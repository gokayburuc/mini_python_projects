# EXTRACT URLS FROM SITEMAP.XML WITH PYTHON 


## REQUIREMENTS 

- xmltodict 

```bash
pip install xmltodict
```

## USAGE 

Run your code using : 

```bash 
python3 bs_extractor.py
```

Write a proper .XML URL such as : 

```bash 
https://www.themajesty.xyz/sitemap.xml 
```

Check your sitemap.csv file 

```bash 
cat sitemap.csv 
```
